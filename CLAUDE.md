# MASQUERADE · 프로젝트 컨텍스트

## 무엇을 만드는가

**MASQUERADE · 가면의 무도회** — 오리지널 컨셉 (실존 게임 아님) · 세 프로젝트 (Sagrada / RandomBattle-Challengers / Faraway) 의 DNA를 조합한 새 카드게임.

**상태**: 디자인 단계 (룰 v3 검증 완료 · 시각/스토리 디자인 진행 중)
**단계**: 아직 코드 없음. 디자인 바이블 먼저 · 그 후 Vite 스캐폴딩.

## 세 개의 필독 문서

1. **`docs/GAMEPLAN.md`** — 게임 룰 · 시스템 (RPS-5, 챌린지, 이벤트, 속삭임 등)
2. **`docs/WORLD.md`** — 세계관 · 스토리 · 5정체 세력
3. **`docs/CHARACTERS.md`** — 40명 손님 카드 상세 (이름 · 정체 · 배경)
4. **`docs/UX.md`** — 씬 흐름 · 마이크로 인터랙션 · 애니메이션

## 세계관 결정 사항

- **시대감성**: 1780년대 베네치아 카니발 + 유럽 궁정 혼합
- **아트 방향**: 수채화 + 부드러운 파스텔 (Faraway와 유사 톤이되 베네치아 요소)
- **이름 스타일**: 혼합 — 본명 (가스통 자작) + 그림자명 (젊은 늑대)
- **컬러 팔레트** (수정됨):
  - `#f5efe3` 크림 (배경)
  - `#e8d5b7` 옅은 골드
  - `#c48b6e` 노을 오렌지
  - `#a68ba8` 라벤더 (신비주의 · 마스크)
  - `#7ba3c8` 카날 블루 (베네치아 물)
  - `#c48b95` 로즈 (왕족)

  - `#5b4636` 딥 코코아 (텍스트)

## 게임 요약 (한 페이지)

**한 줄**: 하나의 왕궁 8자리, 두 사람이 face-down 커밋으로 자리 다툼. 5라운드에 정체 공개 · 8라운드 후 채점.

**시그니처 순간**:
- R3 왕궁 이벤트 (왕이 도착 · 도둑 검거 등)
- R5 GRAND UNMASKING (정체 리빌 + 유예 결투 정산)
- R7-R8 챌린지 페이즈 (점유된 자리 공격)
- RECKONING 세리모니

**핵심 시스템** (v3 · 시뮬 검증됨):
- 공유 왕궁 8자리
- 정체 5종 RPS-5 (왕족/도둑/학자/사냥꾼/신비주의자)
- Face-down 커밋 (카드+자리)
- 결투 유예 → UNMASKING 정산
- 챌린지 (R7-R8만)
- 자리별 성격 (Slot Buffs)
- 왕궁 이벤트 (R3, R6)
- 속삭임 카드 (15%)
- 가면 교체 (R3 옵션)
- 동맹 (같은 정체 결투)
- 추방석 페널티 (-4/장, 5장 시 몰락)
