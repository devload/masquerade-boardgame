# masquerade-boardgame

**MASQUERADE · 가면의 무도회** — 오리지널 카드게임 컨셉 (실존 게임 아님).
Sagrada + Challengers + Faraway 세 프로젝트의 DNA 조합.

**상태**: 디자인 단계 (룰 v3 검증 완료 · 시각/스토리 진행 중 · 아직 코드 없음)

## 한 줄 소개

하나의 왕궁 8자리, 두 사람이 face-down 커밋으로 자리 다툼. 5라운드에 정체 공개 · 8라운드 후 채점.

## 세계관

**1780년대 베네치아 카니발 + 유럽 궁정 감성**. 왕궁 관습에 따라 자정에 가면이 벗겨지고 진짜 정체가 드러난다.

## 감성 & 아트

수채화 + 부드러운 파스텔. 크림 · 라벤더 · 노을 오렌지 · 카날 블루. Faraway와 톤은 유사하되 베네치아 특유의 화려함과 미스터리 강조.
