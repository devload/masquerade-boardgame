# MASQUERADE · Image Generation Prompts

Full art asset prompt catalog for the **Il Ballo dei Volti Argentati** — Ombra Kingdom's Grand Masquerade (1780s Venetian court).

Covers **40 guest portraits**, **8 round-table seats**, **3 rival directors**, **6 court events**, and **3 signature scenes** — 60 prompts total, all tuned for a consistent watercolor + soft-pastel painterly language reminiscent of Faraway boardgame illustrations, elevated with Venetian carnival mystery.

---

## Universal Style Prefix

Paste this block at the **top of every prompt** for a coherent house style. Adjust only the subject block after it.

```
Watercolor illustration in soft pastel technique, painterly and
muted — reminiscent of Faraway boardgame art but with Venetian
carnival mystery. 1780s Ombra Kingdom aesthetic: European court +
Venetian carnival. Palette anchored on cream (#f5efe3), warm gold
(#e8d5b7), deep rose (#8b3a4a), lavender (#a68ba8), canal blue
(#7ba3c8), sage green (#88a065). Delicate wet-on-wet washes, gentle
color bleeds, visible paper grain, no hard outlines. Candlelit
atmosphere with soft golden rim light. Painterly, dreamy, refined,
elegant. Subject centered, atmospheric background blur, gentle
vignette. Portrait aspect 3:4 unless otherwise stated.
```

---

## Universal Negative Prompt

Reject with this negative block on every generation to preserve style unity.

```
Photorealism, photographic, hyperreal, 3D render, CGI, plastic
skin, harsh directional lighting, flash photography, modern
clothing (jeans, t-shirts, sneakers, hoodies), contemporary
setting (offices, cars, streetlights, smartphones), 21st century
architecture, anime, manga, chibi, cartoon outlines, comic book
inking, cel shading, glitch art, cyberpunk, neon signs. No bare
face without mask on any guest. No wrong century (no medieval
armor, no Victorian bustles, no Rococo excess past 1790). No
cluttered composition. No text, no watermarks, no logos, no
signatures, no UI elements. No blurry mush, no over-saturated
neon, no muddy dark blacks. Avoid asymmetric distorted faces,
extra limbs, extra fingers.
```

# 40 Guest Portraits

**Whisper (✦)**: subtle magical/mystical secondary element woven in (soft ghost-light, floating symbol, faint aura, etc.)

# Generation Tips

## Recommended Tools

| Tool | Strengths | Notes for this project |
|---|---|---|
| **Midjourney v6 / v6.1** | Best painterly watercolor default; strong style-consistency via `--sref` | Use `--style raw` for less over-styling. `--ar 3:4` for portraits, `--ar 16:9` for scenes. Pin a style-reference image once you have 1–2 hero portraits you like. |
| **DALL-E 3 (via ChatGPT / API)** | Best at following long structured prompts and character descriptions | Best for the guest 60–100 word prompts. Use one reference portrait per identity family to lock down mask style. |
| **Stable Diffusion (SDXL / SD3) + Venetian LoRA** | Full local control, seed reuse, batch runs | Load a Venetian-carnival LoRA + a watercolor LoRA (0.6–0.8 weight each). Seed reuse across a family is the fastest path to consistency. |
| **Ideogram v2** | Good at painterly consistency with a `style_v2` preset | Fallback if MJ / DALL-E is unavailable. |

## Aspect Ratios per Usage

| Asset | Aspect | Reason |
|---|---|---|
| Guest portrait cards | **3:4** | Matches physical card feel and mobile card display |
| Round-table seat art | **3:4** | Pairs with card sizing in seat select UI |
| Rival director portraits | **3:4** | Bot avatar in lobby |
| Court event scenes | **16:9** | Wide banner across R-phase transitions |
| Lobby splash | **9:16** (portrait) or **3:4** | Mobile-first splash |
| Court centerpiece | **16:9** | Match-scene establishing shot |
| UNMASKING cutscene | **16:9** primary, **3:4** crop for card grid | Reveal moment hero |

## Iteration for Consistency

1. **Generate 2–3 "anchor" portraits first** — one Royalty, one Thief, one Mystic — until you love the mask style, palette, and painterly softness. These become your reference.
2. **In Midjourney**, use `--sref <URL>` with those anchors when generating the rest of that family. `--sw 100` for medium style adherence, `--sw 200+` for tighter lock.
3. **In DALL-E 3**, paste the anchor + guest prompt in the same conversation and say "same painterly watercolor style as the previous image, new subject: ..." — this preserves style within a session.
4. **In SDXL**, reuse the same seed (or seed ± 1–10) with a fixed style LoRA weight across a whole identity family. Vary only the subject block.
5. **Batch by identity family**, not by card number — this keeps mask style coherent within Royalty, then Thief, etc. Do all 8 in one session before switching families.
6. **Palette check**: overlay a swatch strip (`#f5efe3 #e8d5b7 #8b3a4a #a68ba8 #7ba3c8 #88a065`) beside outputs; regenerate if the piece drifts into modern-illustration saturated colors.
7. **Mask-style check**: every guest MUST have a full ornate Venetian mask (not just an eye mask). If a generation produces a bare or minimal mask, discard.
8. **Whisper ✦ check**: whisper-flagged guests should have one subtle magical secondary element (ghost light, floating symbol, faint aura). If missing, regenerate with emphasis on the whisper element.

│   ├── guest_03_malcolm_gray_lion.png

Rules:
- Lowercase, `snake_case`.
- Guest files: `guest_<zero-padded-2-digit-id>_<shortslug>.png` — slug is a 2-word English distillation of the shadow-name.
- Keep one master `.png` (2048px long edge) + a `_web.webp` variant (1024px long edge, quality 85) for the client.
- Store any style-reference anchor images used at `art/references/` — never delete them; regenerations depend on them.
- If a card gets multiple takes, suffix `_v2`, `_v3`. Never overwrite the accepted "canon" version.
