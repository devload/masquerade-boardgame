## 1. LOBBY · 초대장

**목적**: 게임 시작 · 상대 봇 선택 · 세션 옵션.

**화면 구성 (top-to-bottom)**:
1. **왕궁 정문 배경** (수채화 · Ombra 왕궁 · 촛불 흔들림)
2. **큰 초대장 카드** (파치먼트 스타일 · 카니발 마스크 로고)
   - "Il Ballo dei Volti Argentati" (이탈리아어)
   - 부제: "가면 뒤 평등의 무도회 · Ombra 왕궁 · 800년"
3. **CTA "무도회 입장"** — 크림/골드 외곽 · 커다란 CTA
4. **상대 선택** (3장 카드 형태):
   - 말콤 백작 (초급)
   - 이자벨라 부인 (중급)
   - 그림자 (고급)
5. **하단 배지**: 이전 세션 기록, 도감 진행률 (v2)

**애니메이션**:
- 왕궁 정문 창문에서 촛불 은은한 흔들림
- 앰비언트 dust particles (Faraway 스타일이되 골드 톤)
- 초대장 카드 subtle 호흡 (breath)
- 배경 음악: 미뉴에트 하프시코드 은은하게

## 2. COURT · 매치 씬 (핵심)

**목적**: 매 라운드 face-down 커밋 · 결투 판정 · 정보 확인.

### 화면 구성

**Top (헤더 · 40px)**
- 라운드 표시 · 도트 바 R1/8 → R8/8
- 라이브 스코어보드 · 나 vs 상대 (실시간)
- R5 UNMASKING 카운트다운 (전반부만)
- 이벤트 인디케이터 (R3, R6 후 활성 이벤트 명칭)

**Middle · 원탁 (핵심 · 350px)**
- 8자리 **원형 배치** (실제 원탁처럼) — 화면 중앙 · 회전 가능
- 각 자리에 이름 라벨 (Trono · Vicolo · Biblioteca · Foresta · Cuore · Sfera · Accademia · Fianco)
- **자리 버프 힌트** — 라벨 옆에 유리 정체 아이콘 (예: Slot 1 옆에 👑+5)
- 자리 상태:
  - 비어있음: 파치먼트 색 · 빈 좌석
  - 내 카드 놓임: 골드 테두리 · 마스크 카드
  - 상대 카드 놓임: 실버 테두리 · 마스크 카드
  - 유예 결투: 붉은 pulsing 오라 + 두 카드 face-down 스택
  - UNMASKING 후: 마스크 벗겨진 상태 · 정체 아이콘 오라

**Middle-bottom · 컨텍스트 (40px)**
- 현재 상태 안내 텍스트
  - 전반부: "가면 뒤 자리 다툼 · 정체 미공개"
  - UNMASKING 후: "정체 밝혀짐 · 자정 이후"
  - R7-R8: "챌린지 가능 · 점유된 자리 공격"

**Bottom · 손패 & 커밋 UI (200px)**
- 3장 손패 (좌우 스와이프 or 탭)
- 각 카드: 이름 + 표면 점수 + 아이콘 + (전반부) 마스크 (후반부) 정체 표시
- 카드 선택 시 골드 링 활성

**Bottom footer · 커밋 CTA (100px)**
- **자리 선택 인디케이터**: 8자리 미니 도트 (선택된 자리 골드)
- **커밋 버튼** "봉인 (Face-down)" — 카드+자리 확정 시 활성

**우측 사이드 · 컨텍스트 패널 (60px 접힘 → 200px 펼침)**
- 추방석 표시 (내 · 상대)
- 속삭임 정보 로그
- 왕궁 이벤트 목록
- 콤보 진행 힌트

### 마이크로 인터랙션

**손패 카드 탭**
- 카드 스케일 1.05 · 골드 링
- 다른 카드 스케일 0.95
- 100ms 트랜지션

**자리 탭 (자리 선택)**
- 자리 골드 링 pulsing
- 다른 자리 무반응
- 자리 라벨 확대

**커밋 (봉인)**
- 카드 손패에서 자리로 슬라이드 · face-down 회전
- 자리 스포트라이트 잠시 켜짐
- **양쪽 봉인 완료** → 화면 어두워지고 리빌 시퀀스

**리빌 시퀀스** (봉인 후 자동)
- 2장 face-down 카드가 동시에 flip (rotateY 180 → 0)
- 옵션 A · 다른 자리: 카드 각자 자리에 안착 (파치먼트 색으로 안정)
- 옵션 B · 같은 자리: 두 카드가 자리에서 겹침 · **⚔ 결투 유예** 배너 팝

**속삭임 트리거** (배치 시 15% 확률)
- 카드에서 반짝이 오라 → 상대 손패로 향함
- 우측 로그에 "속삭임! 상대 손패 🗡 도둑 있음" 알림
- 소리: 부드러운 종소리 + 속삭이는 음성 샘플

**왕궁 이벤트 배너 (R3, R6)**
- 화면 상단에서 슬라이드 다운
- 예: "🎪 가면 축제! 모든 카드 +1"
- 3초 후 상단에 소형 인디케이터로 이동 · 남은 라운드 표시

**가면 교체 페이즈 (R3, 30% 발동)**
- 화면 중앙에 팝업
- "가면을 바꿀 손패 카드를 선택하세요 (스킵 가능)"
- 카드 선택 → 정체 5종 아이콘 원형 팔레트
- 선택 시 카드 정체 스왑 애니메이션 (마스크가 회전하며 다른 색으로)

### 챌린지 UI (R7-R8만)

**활성화 시점**: R7 시작 시 자동. 챌린지 UI 튜토리얼 팝업.

**시각적 표시**:
- 상대 점유 자리들에 **붉은 챌린지 링** 나타남
- 자리 선택 시 챌린지 자리는 붉은 확인 다이얼로그
- "챌린지 · 이 카드로 상대 [카드명]에게 도전? · 실패 시 추방"

## 3. UNMASKING · R5 컷씬 (10초 시그니처)

**시퀀스** (자동 재생):

### Phase 1 · 정체 리빌 (4초)
```
0.0s: 화면 어두워짐 (opacity 1 → 0.2)
0.5s: 원탁 위 촛불 8개 켜짐 (스포트라이트 하나씩)
1.0s: Slot 1 마스크 회전 벗겨짐 · 정체 아이콘 페이드-인 · 오라 색상 발현
1.5s: Slot 2 (같은 방식)
2.0s: Slot 3
2.5s: Slot 4
3.0s: Slot 5 (유예 결투 있으면 두 카드 다 리빌)
3.5s: Slot 6
4.0s: 모든 자리 정체 공개 완료
```

**BGM**: 종소리 12번 (자정) + 크레센도 하프시코드

### Phase 2 · 유예 결투 정산 (4초)
```
4.0s: "유예 결투 정산" 배너 페이드 인
4.5s: 첫 결투 슬롯 확대 · RPS-5 매치 시각화
5.0s: 승자 카드 자리 확보 (오라 강화) · 패자 카드 추방석으로 슬라이드
5.5s: 다음 결투 슬롯 (있으면)
7.5s: 모든 유예 결투 정산 완료
```

**동맹 특수 애니메이션** (같은 정체 결투 시):
- 두 카드 자리에 유지 · 서로 마주보게 회전
- 골드 링 하트 이모지 잠시 반짝 (실제로는 우아한 하트 flourish)
- "🤝 동맹" 배너

### Phase 3 · 후반부 진입 알림 (2초)
```
8.0s: "자정 이후 · 챌린지 개방" 배너
8.5s: 시장 카드들이 정체 공개 상태로 변신 (마스크 사라짐)
9.5s: 화면 밝기 복귀 · COURT 씬으로 자연 전환
```

## 4. RECKONING · 채점 세리모니

**목적**: 최종 점수 계산 · 정체 상호작용 시각화.

**흐름**: 왼쪽부터 오른쪽 순차 스캔

### 화면 구성
**Top**: 라이브 스코어보드 (실시간 카운트업)
**Middle**: 원탁 (자리별 스포트라이트)
**Bottom**: 상세 판정 패널 (현재 채점 카드)
**우측 사이드**: 콤보 진행 · 왕궁 이벤트 · 추방석

### 각 자리 처리 (1.5초씩 · 총 12초)

**Slot N 처리 시퀀스**:
```
0.0s: 자리 스포트라이트 켜짐 · 다른 자리 dim
0.2s: 표면 점수 표시 (골드 팝) — 예: "+3"
0.4s: 이벤트 보너스 (해당 시) — "+1 가면 축제"
0.6s: 자리 버프 (해당 시) — "+5 왕좌 자리"
0.8s: 정체 규칙 발동
       - 왕족: 왕궁 왕족 카운트 → 팝
       - 도둑: 인접 상대 카드 훔침 애니메이션 (붉은 화살표)
       - 학자: 슬롯 매치 시 golden ✓
       - 사냥꾼: 인접 아이콘 매치 하이라이트
       - 신비주의자: 정체 종류 카운트 팝
1.0s: 소계 팝 (예: "+12")
1.3s: 총점 실시간 카운트업 (스코어보드 반영)
1.5s: 다음 자리로
```

**동맹 자리 특수**:
- 자리에 두 카드 모두 표시
- 표면 점수 반반 나눔 (양쪽 스코어보드에 +/-)
- 정체 규칙 발동은 양쪽 다 발동

### 추방석 정산 (3초)
```
0.0s: "추방석 정산" 배너
0.5s: 내 추방석 카드 하나씩 fade 붉은색
1.0s: -4 페널티 팝 · 내 총점 감소 애니메이션
2.0s: 상대 추방석 같은 방식
3.0s: 완료
```

### 콤보 판정 (3초)
```
0.0s: "콤보 판정" 배너
0.5s: 왕궁 지배 (6+ 자리) → 골드 배너 팝 · +15
1.5s: 가문의 위엄 (한 정체 4명+) → 팝 · +15
2.0s: 완벽한 무도회 (5정체 다) → 팝 · 양쪽 +10
2.5s: 평화의 무도회 (결투 0회) → 팝 · 양쪽 +20
3.0s: 완료
```

### 최종 판정 (5초)
- 큰 배너: "승리 (Vittoria)" or "패배 (Sconfitta)" or "무승부"
- 궁정의 왕이 손을 들어 심판하는 애니메이션 (실루엣)
- 종소리 크레센도
- 세션 요약 (결투 승률, 추방 수, 콤보 목록)

## 5. RESULT · 세션 요약

**목적**: 세션 리뷰 · 재도전 or 로비.

**화면 구성**:
1. **결과 배너** (Vittoria · Sconfitta · Pareggio)
2. **최종 스코어** (양쪽)
3. **세션 하이라이트** (자동 생성):
   - "R5에 UNMASKING 유예 결투 3회 발생 · 극적 순간"
   - "챌린지 승률 75%"
   - "왕좌 자리 지배"
   - "속삭임 3회 활용 · 정보전 승리"
4. **콤보 배지** (달성한 콤보 표시)
5. **CTA**:
   - "다음 무도회 (다시 시작)"
   - "로비로"
   - "리플레이 저장" (v2)

**애니메이션**:
- 왕궁 정문에서 손님 실루엣들이 걸어 나가는 은은한 배경
- 촛불 여운

## 6. 마이크로 인터랙션 총정리

### 애니메이션 타이밍 원칙
- **150-300ms**: 대부분의 hover/tap 반응
- **400-600ms**: 카드 flip · 자리 전환
- **1-2초**: 스포트라이트 · 결투 판정
- **3-10초**: 시그니처 컷씬 (UNMASKING · RECKONING)

### 트랜지션 커브
- Ease-in-out for card movements
- Spring physics (damping 20, stiffness 300) for placement/reveals
- Linear for scoreboards countup

### 색상 활용
- **골드** (`#e8d5b7`, `#c48b6e`): 성공, 내 것, 왕족
- **실버** (`#c0c0c0`): 상대 것, 학자
- **딥 로즈** (`#8b3a4a`): 결투, 추방, 도둑
- **라벤더** (`#a68ba8`): 마스크, 신비주의
- **세이지** (`#88a065`): 사냥꾼, 자연
- **크림** (`#f5efe3`): 배경, 파치먼트

### 사운드 디자인
- **미뉴에트** (배경음악): 하프시코드 + 현악 loop
- **카드 놓기**: 벨벳 스치는 소리
- **결투**: 검 부딪히는 소리 + 청동 종
- **속삭임**: 부드러운 종소리 + 속삭이는 음성 샘플
- **UNMASKING**: 자정 종 12번 + 크레센도
- **몰락**: 낮은 붐 + 깨진 유리
- **RECKONING**: 하프시코드 아르페지오
- **콤보 배너**: 팡파레

### 접근성
- 모든 애니메이션 스킵 가능
- 색맹 대응: 정체 아이콘 명확 (색상만 의존 X)
- 큰 터치 타겟 (44px 이상)
- 시각 힌트 텍스트도 함께 (스크린리더 대응)

## 상세 씬 스토리보드 (예시)

### 시나리오: R5 UNMASKING (신참 플레이어 관점)

**R1 시점**:
- 손패 3장 다 마스크 (미스터리)
- 자리 라벨 봄: "Trono · 왕족 +5", "Cuore del Ballo · 사냥꾼 +2 & 인접 2배"
- 자리 골라야 하는데 표면 점수만 힌트

**R2-R4**:
- 매 라운드 커밋 → 리빌 → 결투 유예 or 배치
- 우측 로그: "R3: 왕궁 이벤트 · 가면 축제"
- R4에 UNMASKING 예고 배너 pulsing

**R5 UNMASKING 순간**:
- 화면 완전히 어두워짐 · 심장이 뛴다
- 종소리 12번 · 각 카드 마스크가 회전하며 벗겨짐
- **"어? 저 카드가 왕족이었다니!"** — 순간의 카타르시스
- 유예 결투 정산 시 붉은 검 화살표 · **내 도둑이 상대 왕족 이김**
- 상대 추방석에 그 카드 추가 · "-4" 팝
- 후반부 진입 · 시장이 정체 공개 상태

**R6-R8 후반부**:
- 이제 정체 아는 상태 · 카운터 배치 시작
- R7에 챌린지 UI 활성 · 붉은 링 등장
- **"이 상대의 학자를 내 도둑이 챌린지!"** — 결정적 순간

**RECKONING**:
- 왼쪽부터 채점 스캔 · 각 자리 스포트라이트
- 도둑 훔침 · 학자 매치 · 사냥꾼 인접
- 콤보 판정 · "가문의 위엄! +15"
- 최종 45 vs 32 · 승리!
